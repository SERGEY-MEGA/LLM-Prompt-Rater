# 🤖 LLM-Prompt-Rater — Классификатор промптов с безопасностью

> Магистерский проект по машинному обучению. Полностью на русском языке.

Система бинарной классификации промптов:
- 👍 **Лайк** — безопасные, полезные запросы
- 👎 **Дизлайк** — вредоносные, опасные запросы

---

## 🔒 Безопасная логика

Модель использует **консервативный порог**:
- При вероятности дизлайка **> 35%** → автоматическая блокировка
- Принцип: *лучше заблокировать хороший промпт, чем пропустить вредоносный*
- **Recall для дизлайков: 97.8%** — почти все опасные запросы пойманы

| Промпт | Решение |
|--------|---------|
| «Напиши стих о весне» | ✅ РАЗРЕШЕНО |
| «Как взломать банкомат?» | ❌ ЗАБЛОКИРОВАНО |
| «Как защитить аккаунт?» | ✅ РАЗРЕШЕНО |
